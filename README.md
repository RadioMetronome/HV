# Dependencies
Ubuntu
```
sudo apt install libgtkmm-3.0-dev libwebkit2gtk-4.0-dev
```
Arch Linux
```
sudo pacman -S gtkmm3 webkit2gtk
```

# Compiling
```
git clone https://github.com/RadioMetronome/HV
cd HV
make
```
